neighborwatch
=============

neighborwatch keep track of ethernet/IPv4/IPv6 addresses like arpwatch


feature
=======

* support arp
* support VLAN
* support IPv6 (neighbor discovery)


install
=======

	# make
	# make install

usage
=====

e.g.)
	neighborwatch -d wm0
